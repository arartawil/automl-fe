# Examples

This directory contains example scripts demonstrating AutoML-FE usage.

## Planned Examples

- **Basic Usage**: Simple optimization with tracking
- **Custom Algorithms**: Wrapping custom evolutionary algorithms
- **Visualization**: Generating explainability plots
- **Comparison**: Comparing different optimization methods
- **Real-world**: Practical optimization problems

## Running Examples

Each example is self-contained and can be run directly:

```bash
python examples/basic_usage.py
```

Stay tuned for more examples!
