# Documentation

This directory will contain comprehensive documentation for XAIevo.

## Planned Documentation

- **User Guide**: Getting started, tutorials, examples
- **API Reference**: Detailed API documentation
- **Theory**: Explainability methods and algorithms
- **Examples**: Real-world use cases

## Building Documentation

Documentation will be built using Sphinx (coming soon).

```bash
# Install documentation dependencies
pip install sphinx sphinx-rtd-theme

# Build documentation
cd docs
make html
```
